This portfolio is built using:
- **HTML** (59.4%)
- **CSS** (33.7%)
- **JavaScript** (6.9%)

It highlights my passion for clean design, responsive layouts, and interactive user experiences.

## Features

- Responsive design for all devices
- Clean and modern UI
- Project gallery and descriptions
- Contact section

## Getting Started

To view the portfolio locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Mohammedalswadi/personal-protfolio.git
   cd personal-protfolio
   ```
2. Open `index.html` in your web browser.

## Folder Structure

- `/`: Main HTML files
- `/css`: Stylesheets
- `/js`: JavaScript files
- `/images`: Project images and assets
